
# Number Guessing Game (Java)
📌 Project Overview

The Number Guessing Game is a simple console-based Java application developed using Core Java (J2SE). The program generates a random number between 1 and 100, and the player tries to guess it. The game provides hints after each incorrect guess and allows unlimited attempts until the correct number is guessed.

After each game, the player can choose to play again or exit the application.

✨ Features
Generates a random number between 1 and 100
Unlimited guessing attempts
Displays hints:
Too High! Try Again.
Too Low! Try Again.
Counts the total number of attempts
Displays a performance message based on the number of attempts
Option to play multiple rounds
Input validation for numbers outside the valid range
Beginner-friendly and easy-to-understand code

▶️ How to Run
Compile

Run

🎮 How to Play
Run the program.
Enter a number between 1 and 100.
The program will tell you whether your guess is too high or too low.
Continue guessing until you find the correct number.
After winning, the program displays:
Correct number
Total attempts
Performance message
Choose Y to play again or N to exit.
